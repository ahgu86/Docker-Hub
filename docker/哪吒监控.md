### Docker部署哪吒监控

`docker-compose.yml`示例配置

```
services:
  nezha:
    image: ghcr.io/naiba/nezha-dashboard:v0.20.13
    container_name: nezha
    restart: always
    ports:
      - "8008:80"   # 面板端口
      - "5555:5555" # 监控端口
    volumes:
      - ./config.yaml:/dashboard/data/config.yaml   # 配置文件
      - ./theme-daynight:/dashboard/resource/template
```
